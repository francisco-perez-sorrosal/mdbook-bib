# Introduction to Chicago Author-Date Style

This test book demonstrates the **Chicago Author-Date citation style** using mdbook-bib's CSL backend powered by hayagriva.

## What is Chicago Author-Date Style?

The Chicago Author-Date style uses **in-text citations** with the author's last name and year of publication, like this: {{#cite watson1953}}.

## Key Features

Chicago Author-Date style is characterized by:
- **Parenthetical citations** in the text (e.g., (Watson and Crick 1953))
- **Author and year** format
- **Alphabetical ordering** in the reference list by author's last name
- Commonly used in social sciences, humanities, and natural sciences

## Example Citations

When citing multiple works, they appear together {{#cite shannon1948}} {{#cite knuth1984}}.

The shorthand syntax also works: @@turing1950 and @@lamport1986.

## Bibliography Organization

In Chicago Author-Date style, the reference list is alphabetized by author surname, making it easy to locate sources.
